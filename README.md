## Usage:
1. Create a new folder `.streamlit` and create a file titled `config.toml` within it. Save this file with the following content:
```
[theme]
primaryColor="#2d4be0"
backgroundColor="#f9f4f4"
secondaryBackgroundColor="#e4edff"
textColor="#3a6ae4"
```
